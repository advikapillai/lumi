# Lumi

Lumi is a skincare recommendation web application built with Flask that generates personalized skincare routines based on a user's skin type, primary concern and budget.

## Features

* Personalized morning and evening skincare routines
* Product recommendations based on budget
* Animated pastel interface with floating illustrations
* Interactive product links
* Responsive, calming UI

## Built With

* Python
* Flask
* HTML
* CSS
* JavaScript

## Future Improvements

* User accounts
* Saved skincare routines
* Skin progress tracker
* Ingredient checker
* Product ratings and reviews
* AI implemented skin analysis
